## Project Overview

**ConcreteAI** is a lightweight, browser-based AI webpage that predicts the compressive strength of concrete based on mix proportions.  
It uses a **trained model** (developed using Python) and implements the prediction logic in JavaScript for real-time results in the browser.  
The output is displayed in a **glowing result box** with a clean and modern AI-inspired design.

The project emphasizes **modern UI design** with:
- Black background  
- White bordered forms  
- Glowing result output  
- Clean, two-column layout for input data  

---

## Features

- AI-powered prediction of concrete strength (MPa) instantly  
- Real-time predictions using a trained Python model implemented in JavaScript  
- Modern, responsive UI with glowing effects  
- Easy-to-use form with labels and placeholders  
- Works entirely in the browser (no backend required)  

---

## Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript (ES6)  
- **Model Training:** Python
